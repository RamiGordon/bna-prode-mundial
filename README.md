# Prode Mundial 2026

PWA para jugar al prode del Mundial 2026 entre amigos. Stack: Next.js (latest) + Supabase + Vercel.

## Setup local

```bash
# 1. Instalar deps
npm install

# 2. Variables de entorno
cp .env.example .env.local
# Completar con las credenciales de Supabase

# 3. Levantar dev server
npm run dev
```
